# Therian Web

Web mobile-first sobre therianthropy, optimizada para monetización y desplegada en Cloudflare Pages.

## Desarrollo
Este proyecto consiste en archivos estáticos. 
- Puedes abrir `index.html` en tu navegador.
- Si instalas dependencias o usas un bundler, añade los scripts correspondientes.

## Despliegue (Cloudflare Pages)
Configura Cloudflare Pages para conectarse a este repositorio GitHub, con la rama `main` conectada a producción y despliegues automáticos (Previews) configurados para los Pull Requests.

## Compliance
- Las páginas de privacidad, cookies y aviso legal están enlazadas en el footer y preparadas para el cumplimiento.
- El archivo `ads.txt` está en la raíz para AdSense.
